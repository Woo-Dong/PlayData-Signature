## Docker-compose Setting 
---
#### 1. git repository Clone
```
$ git clone 
$ cd PlayData-Signature
$ cd web_server
```

#### 2. Docker-compose up 
```
$ docker-compose up -d --build
```

#### 3. Visit Web Site on your localhost
